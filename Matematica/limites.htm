Límites(funciones de una variable)

Límites en infinito:

Se puede definir el límite cuando x tiende a infinito de una función como el valor al cual se acerca la función cuando tomamos valores muy grandes de x así también se puede definir el limite cuando x tiende a menos infinito como el valor al cual se acerca a la imagen de la función para x negativos de modulo muy grande. Tomemos como ejemplo la función 1/x si consideramos valores de x muy grandes sean estos positivos o negativos la función toma valores cada vez mas chicos mientras más grande sea el valor de x mas chica será la imagen de 1/x, es decir la imagen se acercara cada vez mas a cero (tendera a cero). Entonces decimos que el límite cuando x tiende a ∞ de 1/x es 0. Miremos el gráfico a continuación para aclarar las ideas.

En el gráfico se puede ver como para x grandes la curva se acerca a cero. Sin embargo notar que para x → +∞ la función se acerca a cero siempre por valores mayores a cero esto se nota 0+, y para x → -∞ se acerca a cero por los valores menores que cero esto se nota 0-

Ahora uno podría generalizar este resultado, si calculamos el límite cuando x tiende a ∞ de una función que se puede escribir como una constante sobre algo que toma valores arbitrariamente grandes (tiende a infinito) diremos que esta función tiende a cero ya que el cociente para x grande se hace arbitrariamente pequeño.

Consideremos por ejemplo 2 x es una función creciente, cuando tomamos valores grandes de x la función tiende a +∞ con esto queremos decir que la imagen de 2 x alcanza valores arbitrariamente grandes para x grande. Ahora si consideramos la función 1/2 x veremos que estamos haciendo el cociente entre un numero constante y una función que toma valores cada vez más grandes cuanto mayor sea el valor de x, entonces el límite cuando x tiende a mas infinito de 1/ 2 x es cero.

Definición límite en infinito positivo:

Límite cuando x → +∞ de f(x) es igual a b si y solo si dado un ε mayor que cero existe M positivo tal que si x > M entonces : |f(M)-b| ‹ ε

Definición límite en infinito negativo:

Límite cuando x → -∞ de f(x) es igual a b si y solo si dado un ε mayor que cero existe M positivo talque si x < -M entonces |f(M)-b| ‹ ε

Ejemplo de cálculo de límite por definición:

Demostrar que el límite cuando x → +∞ de f(x) = 2+3/x es 2.

Queremos ver que existe un M tal que se satisfaga la definición para un epsilon arbitrario. Por lo tanto tenemos que tratar de acotar

|2+3/x-2|

Primero cabe observar que se pueden cancelar los 2 resultando

|3/x|

Ahora

|3/x| < 3/M

Si pedimos que sea menor a epsilon

|3/x| < 3/M < ε

Entonces basta con elegir M > 3/ε para que se satisfaga la condición anterior

Límites en un Punto:

Se puede definir el límite cuando x tiende a un número cualquiera de una función como el valor al cual se acerca la misma cuando tomamos valores muy cercanos a este número de x .Ahora a diferencia con los límites en infinito, uno se puede acercar a un punto por números mayores a él o menores y la función no tiene por que tener el mismo comportamiento en ambos casos (A estos se los llaman límites laterales). Tomemos como ejempló la función 1/x si consideramos valores de x cercanos a 0 la función toma valores muy grandes entonces se dice que tiende a infinito como se puede ver en la sección anterior. Pero hay que distinguir los casos en los que nos acercamos a 0 por números menores a él y los que nos acercamos por números mayores que él, en un caso la función toma valores muy grandes positivos y en el otro negativos por lo tanto tiende a mas o menos infinito según por donde nos acerquemos. Entonces uno no debería hablar de límites sino de límites laterales en un punto. Pero en muchos casos los límites laterales son iguales, cuando esto ocurre y solo cuando esto ocurre se habla de límite a secas. Como por ejemplo el límite cuando x tiende a 2 de f(x)=3x+7. Se puede observar que cuando x esta cerca de 2, f(x) esta cerca de f(2) es decir de 13, independientemente de si uno se acerca por derecha u izquierda. En muchos casos, como en este el cálculo del límite se reduce entonces a reemplazar el número en la función. Pero esto no siempre es así, hay un funciones que no están definidas en el punto que se quiere calcular el límite como por ejemplo.

Se ve claramente que la función no esta definida en x=1, cuando esto ocurre se dice que el límite es indeterminado. En este caso tanto el numerador como el denominador son cero en x=1 y como no se puede realizar el cociente 0/0 no se puede simplemente reemplazar en la función el punto. Lo que se hace es tratar de mediante operaciones algebraicas llegar a una expresión en la que no aparezca la indeterminación como se hacía en los límites en infinito. Las indeterminaciones mas comunes son.

Es importante que se entienda que cuando hablamos de indeterminaciones estamos hablando de limites, con esto lo que quiero decir es que por ejemplo que algo que tiende a 1 elevado a una potencia que tiende a infinito es indeterminado en principio. Pero si lo que tenemos es el número constante 1 elevado a una potencia que tiende a infinito esto no es una indeterminación pués el número constante 1 elevado a la potencia que sea seguirá siendo 1.

Definición formal de Límites:

Límite cuando x → a de f(x) es igual a b si dado un ε mayor que cero existe un δ tal que si. |x-a| < δ ⇒ |f(x)-b| < ε
